# json-compare
## This project is for comparing two json object.  
### The json exist in file, then read it from file and transfer it to a jsonObject. This project has two dependencies: org.json(http://mvnrepository.com/artifact/org.json/json/20171018) and google.json(http://mvnrepository.com/artifact/com.google.code.gson/gson/2.8.2). Please download these jar by yourself. 
### About effciency, in my two json file comparasion, it wastes about 20 - 40 ms.
